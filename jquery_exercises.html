This paragraph is inside the p tags. ID Selector #id Selects a single element with the given id attribute. Class Selector .class Selects all elements with the given class. Element Selector element Selects all elements with the given tag name. Multiple Selector selector1[, selector2[, ...]] Selects the combined results of all the specified selectors. All Selector * Selects all elements.
